Notion Team plan

Price: $10 per user per month, billed annually. ($12/user/month monthly.)

What's included

  • Unlimited blocks for teams.
  • Permission groups.
  • Admin tools.
  • Page history: 30 days.

Integrations

Native: Slack, GitHub, Jira, Figma, Google Drive, Asana, Linear, Zoom, Tableau. Public API available.

API limits

API rate limit: 3 requests/second per integration. No write quota on the Team plan.
